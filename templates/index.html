📖 About

About NeuroScan

NeuroScan V3.0 is an advanced AI-powered brain tumor detection system built with Python, TensorFlow, and Flask. It uses a deep Convolutional Neural Network (CNN) to classify brain MRI scans as either containing a tumor or being normal.

The model was designed with a 4-block CNN architecture using Batch Normalization, Dropout regularization, and Global Average Pooling. Images are enhanced using CLAHE (Contrast Limited Adaptive Histogram Equalization) before analysis to improve detection accuracy.

This project was built as an educational tool to demonstrate the power of deep learning in medical image analysis. All processing is done locally — your data never leaves your device.
